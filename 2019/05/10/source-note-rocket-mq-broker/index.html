【阅读笔记】rocketmq broker （六）

发表于 2019-05-10 | 更新于 2019-06-15

基本功能

rocketmq 消息队列的主体。producer 将消息发送到 broker；consumer 从 broker 拉取消息消费。broker 使用 store 模块将消息存储到磁盘。

代码分析

BrokerController 表示了 broker。borker 由 BrokerStartup 类引导，并创建 BrokerController 对象，然后执行 BrokerController 的 start 方法启动 broker

BrokerController 成员

clinet 管理（producer / consumer）

客户端管理关注客户端的上线和下线，主要通过 3 个类实现： ProducerManager （管理 producer），ConsumerManager（管理 consumer），ClientHousekeepingService（管理连接）
